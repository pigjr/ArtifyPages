# 艺绘书页 (ArtifyPages)

## 项目简介
艺绘书页是一款专为鸿蒙6.0 PC平台开发的AI电子书插图生成应用。通过先进的AI技术，为电子书自动生成精美的插图，提升阅读体验。

## 应用信息
- **中文名**: 艺绘书页
- **英文名**: ArtifyPages
- **平台**: 鸿蒙6.0 PC
- **主要功能**: AI生成电子书插图

## 技术栈
- 开发框架: ArkTS/ArkUI
- 目标平台: 鸿蒙6.0 PC

## 项目结构
```
ArtifyPages/
├── AppScope/          # 应用作用域配置
├── entry/             # 主模块
├── hvigor/            # 构建工具配置
├── build-profile.json5 # 构建配置
└── oh-package.json5   # 依赖配置
```
